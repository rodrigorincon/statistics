# TESTES DE HIPÓTESES

Existem vários, mas aqui listarei os mais comuns:

- Teste t
- Teste z
- Teste F
- Anova
- Qui-Quadrado
- Pearson
- Correlação
- Análise de Regressão
- Wilcoxon
- Spearman
- Binomial
- Poisson
- Fisher
- McNemar

# QUANDO USAR

- Comparar média (1 média com valor específico ou 2 médias entre si): **Teste T**
- Comparar 3 ou mais médias (de qualquer modo): **Anova**
- Comparar proporção: **Teste Z**
- Comparar vars categórias (ex: antes e depois): **McNemar**
- Relação entre vars categóricas: **Qui-Quadrado**
- Relação entre vars numéricas: **Correlação ou Análise de Regressão**
