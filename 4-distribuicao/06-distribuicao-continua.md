# DISTRIBUIÇÃO PROBABILISTICA CONTINUA

- As distribuições discretas não servem para vars contínuas
- Algumas até tem algumas parecidas, mas não dá pra usar as mesmas equações direto
- A principal é a normal

## Qual Distribuição Usar

- Se for calcular tempo entre eventos: **exponencial**
- Se tem a população toda: **normal**
- Se tem só amostras:
	- amostras pequenas: **T**
	- amostras grandes: **normal**
